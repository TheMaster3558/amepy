Welcome to pyame's documentation!

Getting Started
---------------

You first must install `pyame`

.. code:: sh

    # Linux/MacOS
    python3 -m pip install pyame

    # Windows
    py -3 -m pip install pyame


You must also have an API key, get one `here <https://api.amethyste.moe/>`_
